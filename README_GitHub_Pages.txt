# Publicar en GitHub Pages — versión funcional (sin PWA extra)
Archivos:
- index.html (versión con roles abreviados + iconos, export PDF/Excel/CSV, autosave, etc.)
- .nojekyll

Pasos:
1) Crea un repositorio público en GitHub (ej. `mesas-jl`).
2) Sube **index.html** y **.nojekyll** a la **raíz** del repositorio.
3) En el repo: Settings → Pages → Build and deployment
   - Source: Deploy from a branch
   - Branch: main  /root
4) Abre la URL de Pages (https://TU-USUARIO.github.io/mesas-jl/).
5) (Opcional) iPhone: Compartir → Añadir a pantalla de inicio.
